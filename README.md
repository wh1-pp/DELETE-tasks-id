# Task Manager API

REST API для управления задачами с разграничением прав доступа между администратором и супер-администратором.

## Стек технологий
- Python 3
- Django 6.0
- Django REST Framework 3.17

## Установка и запуск

### 1. Клонируй репозиторий
git clone <ссылка на репозиторий>
cd myproject

### 2. Установи зависимости
pip3 install django djangorestframework

### 3. Примени миграции
python3 manage.py migrate

### 4. Создай суперпользователя
python3 manage.py createsuperuser

Введи имя, email и пароль — запомни их, они понадобятся для тестирования.

### 5. Создай тестовую задачу
python3 manage.py shell

Затем по очереди:
from django.contrib.auth.models import User
from tasks.models import Task
user = User.objects.first()
Task.objects.create(title="Тестовая задача", owner=user)
exit()

### 6. Запусти сервер
python3 manage.py runserver

Сервер запустится по адресу: http://127.0.0.1:8000/

## Эндпоинты

### DELETE /tasks/{id}
Удаление задачи по ID.

**Права доступа:**
| Пользователь | Действие |
|---|---|
| Анонимный | 403 Forbidden — доступ запрещён |
| Администратор | Может удалить только свою задачу |
| Супер-администратор | Может удалить любую задачу |

**Возможные ответы:**
| Код | Описание |
|---|---|
| 204 | Задача успешно удалена |
| 403 | Нет доступа |
| 404 | Задача не найдена |

## Тестирование через Postman

### Установка Postman
Скачай и установи Postman с официального сайта: https://www.postman.com/downloads/

### Как отправить запрос

1. Открой Postman
2. Нажми New → HTTP Request
3. В выпадающем списке выбери метод DELETE
4. В строку URL введи: http://127.0.0.1:8000/tasks/1/
5. Нажми синюю кнопку Send

### Ожидаемые результаты

**Без авторизации:**
- Статус: 403 Forbidden
- Ответ: {"error": "Доступ запрещён"}

**Задача не существует:**
- Статус: 404 Not Found
- Ответ: {"error": "Задача не найдена"}

**Успешное удаление (от супер-администратора):**
- Статус: 204 No Content
- Ответ: пустой
